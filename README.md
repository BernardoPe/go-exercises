# Learning Go

This repository contains exercises and projects implemented while learning the Go programming language.

> **Note:** All projects were created for learning purposes only and include various simplifications. They were not designed for production use.

---

## Projects

### 1. [File Downloader](file-downloader)
A concurrent file downloader using goroutines and channels.

### 2. [KV Store](kv-store)
A simple in-memory key-value store.

### 3. [HTTP Server](http-server)
A simple HTTP 1.1 server implementation.

### 4. [Websocket Chat](ws-chat)
A real-time chat application using WebSockets.
